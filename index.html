Hello World in 5 languages

This document demonstrates "Hello World" applications in some of the most popular languages as of July 2020, according to the TIOBE index.

C(++)

C, and its extension, C++, are in tandem with each other the most popular languages in recent history.

C allows for a complex vocabulary, but doesn't enforce it by introducing verbosity like Java does. How little you can get away with may confuse developers coming from other languages, that make a high level of abstraction part of their design.

This is an example of a Hello World application in C:

                    
#include <stdio.h>
int main() {
    // printf() displays the string inside quotation
    printf("Hello, World!");
    return 0;
}
Java

For decades, Java is going head-to-head with C(++). Its rich ecosystem provides many opportunities for developers to latch on.

Java code is much more verbose than C(++) code and the compiler enforces this verbosity rigorously. This leads to some critique from experienced developers who wish for the support of a more rapid development style. Beginners, however, are met with very clear expectations.

This is an example of a Hello World application in Java:

                    
public class Application {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
Swift

Swift is a programming language for Apple devices and Linux. It is relatively young, being first released in 2014.

In comparison to C(++), Swift provides a less steep learning curve. An active community and a large variety of tools makes it one of the most successful languages of the past decade.

This is an example of a Hello World application in Swift:

                    
import Swift
print("Hello, World!")
Visual Basic

Visual Basic (.NET) is the spiritual successor of the proprietary Microsoft Visual Basic programming language which was no longer being supported after the release of the .NET platform.

Many want it dead but the demand for developers is still steady and VB.NET isn't going to disappear anytime soon. Its importance in legacy systems is undeniable.

This is an example of a Hello World application in Visual Basic (.NET):

                    
Imports System
Module Hello_Program
    Sub Main()
        Console.WriteLine("Hello, World!")
        Console.ReadKey()
    End Sub
End Module
R

The R programming language is designed for statistical computing and a de-facto standard due to its active development for more than two decades.

R has recently seen a significant surge in activity. The language rose from rank 20 in 2019 to rank 8 in 2020.

This is an example of a Hello World application in R:

                    
cat("Hello World\n")
